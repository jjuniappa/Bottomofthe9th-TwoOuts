# 9회말 2아웃

모바일 스와이프로 마지막 타자를 상대하는 브라우저 투구 게임 프로토타입입니다.

## 실행

압축을 풀고 `index.html`을 브라우저에서 열면 됩니다.

## GitHub Pages 배포

1. 저장소 루트에 모든 파일을 업로드합니다.
2. GitHub 저장소의 **Settings → Pages**로 이동합니다.
3. **Deploy from a branch**를 선택합니다.
4. `main` 브랜치와 `/ (root)`를 선택해 저장합니다.

## 조작

- 공을 위로 빠르게 스와이프: 구속과 높이 증가
- 공의 오른쪽에서 시작: 왼쪽으로 휘는 커브
- 공의 왼쪽에서 시작: 오른쪽으로 휘는 커브
- 스와이프 도중 손가락을 좌우로 꺾기: 추가 변화량

이번 버전은 작은 스트라이크존, 커진 공, 높은 입력 감도, 후반에 크게 휘는 cubic Bézier 커브 궤적을 적용했습니다.
