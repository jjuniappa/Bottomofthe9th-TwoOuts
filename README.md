# 9회말 2아웃

모바일 스와이프로 마지막 타자를 잡는 단일 타석 투구 게임 프로토타입입니다.

## 실행

파일을 내려받은 뒤 `index.html`을 브라우저에서 열면 실행됩니다.

로컬 서버 사용 예시:

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000`으로 접속하세요.

## GitHub Pages 배포

1. 새 GitHub 저장소를 만듭니다.
2. 이 폴더의 `index.html`, `styles.css`, `game.js`, `README.md`를 저장소 루트에 업로드합니다.
3. 저장소의 **Settings → Pages**로 이동합니다.
4. **Deploy from a branch**를 선택합니다.
5. `main` 브랜치와 `/root` 폴더를 선택하고 저장합니다.

## 조작

- 야구공을 위로 스와이프하거나 마우스로 드래그합니다.
- 너무 약하면 원바운드가 됩니다.
- 너무 강하면 스트라이크존 위로 빠집니다.
- 공의 오른쪽을 잡고 던지면 왼쪽으로, 왼쪽을 잡고 던지면 오른쪽으로 휩니다.

## 현재 구현

- 모바일 터치 및 마우스 입력
- 스와이프 속도에 따른 높이와 구속
- 접촉 위치에 따른 좌우 커브
- 볼·스트라이크·삼진·볼넷 판정
- 간단한 타자 스윙 및 컨택 확률
- 외부 에셋 없는 Canvas 렌더링
